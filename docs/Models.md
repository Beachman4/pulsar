Models
=====

## Property Definitions

The properties for each model are defined using the `$properties` variable on every model class.

### `type`

The type of the property.

Accepted Types:
- string
- number
- boolean
- date
- array
- object

String, Required

### `mutable`
        Specifies whether the property can be set (mutated)
        Boolean
        Default: true
        Optional
